# License selection required before publication

Maria Smith intends this project to support an open-source science movement.
The existing local corpora do not declare a top-level software license, so this
clean-room scaffold does not guess one.

Before a public repository is created, select and add:

- a software license for source and generated certificates;
- a documentation and paper license;
- an explicit policy for the Ernos Labs designation and attribution; and
- any separate terms required for contributed empirical datasets.

Until that choice is made, the files are local working materials and the phrase
"open source" describes the intended publication model rather than a granted
license.
