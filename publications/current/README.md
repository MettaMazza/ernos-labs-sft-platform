landing page, [`README.md`](../../README.md). This makes the paper—not a folder
listing—the first object a visitor reads.

The archival typeset edition is
[`output/pdf/there-is-no-nothing-sft-open-science-platform.pdf`](../../output/pdf/there-is-no-nothing-sft-open-science-platform.pdf).

branch inventory closes; the final TOE rewrite follows only after every branch
paper closes.
