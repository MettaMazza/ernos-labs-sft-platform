# V3 language policy

## Python is the accessible host

The third reconstruction uses Python because it is widely readable, portable
and easy for independent researchers to run. The authoritative v3 route should
be reproducible with a normal Python installation and the standard library.

Python does not select SFT mathematics. It hosts exact representations,
enumerators, proofs, receipts and interfaces.

## Core rules

- Use `fractions.Fraction` behind guarded SFT types for exact rational values.
- Reject inadmissible semantic values at construction boundaries.
- Represent orientation and complement with labels or branches, not imported
  signed arithmetic.
- Do not use binary floating point in derivations or mathematical certificates.
- Put conventional decimal measurements in explicit empirical adapters.
- Do not let empirical adapter values flow back into derivation.
- Prefer immutable dataclasses, explicit enums and canonical JSON artifacts.
- Prefer `unittest` for the portable baseline.
- Keep optional performance dependencies outside the proof authority.
- No notebook is an authoritative derivation or receipt.

Host-language indices, empty containers, byte offsets and process status codes
are implementation mechanics, not SFT mathematical objects. Their use must
remain behind a documented boundary.

## Independent certificates

Important results should generate portable C certificates or another
independent, readable execution form. Generated code must recompute the result
and controls; it must not merely contain the expected answer.

## Version discipline

The exact Python version used for official evidence will be recorded in every
execution receipt. A minimum supported Python version will be selected after
the first portability census rather than assumed in advance.

## V4 is already registered

Python is deliberately not declared the final SFT-native language. The fourth
reconstruction will be written in SFT-derived, self-hosted language and runtime
principles after the v3 completion gate in
[`docs/V4_SELF_HOSTED_REBUILD.md`](docs/V4_SELF_HOSTED_REBUILD.md) is met.
